# Rule-Based AI Chatbot

## Overview

This project is a simple Rule-Based AI Chatbot developed in Python for DecodeLabs Artificial Intelligence Project 1.

The chatbot interacts with users by responding to predefined questions using if-elif-else statements. It runs continuously until the user enters the exit command.

## Features

- Greets users
- Answers predefined questions
- Handles unknown inputs
- Displays help menu
- Continuous conversation
- Exit command
- Uses only Python control flow

## Technologies Used

- Python 3
- if-elif-else
- while loop
- input() function
- print() function

## How to Run

1. Install Python.
2. Open the project folder.
3. Run

python chatbot.py

## Project Objective

To understand the basics of Artificial Intelligence by creating a chatbot that follows predefined rules without using Machine Learning.
