1. Introduction

Artificial Intelligence (AI) is a branch of computer science that focuses on developing systems capable of performing tasks that normally require human intelligence. One of the simplest applications of AI is a rule-based chatbot. Unlike machine learning models, a rule-based chatbot follows predefined rules to respond to user inputs.

This project demonstrates the basic concepts of Artificial Intelligence by creating a chatbot that interacts with users through simple text-based conversations using Python. The chatbot responds to predefined commands, handles greetings, provides information on selected topics, and exits gracefully when requested.

2. Project Objective

The main objectives of this project are:

To understand the fundamentals of rule-based Artificial Intelligence.
To develop a chatbot using Python.
To implement decision-making using if-elif-else statements.
To create a continuous conversation using a while loop.
To improve user interaction by recognizing different ways of asking the same question.
3. Problem Statement

Many beginner chatbot projects require users to type only exact commands, making interaction limited and less user-friendly. The goal of this project is to develop a chatbot that recognizes multiple variations of predefined questions while maintaining a simple rule-based approach without using Machine Learning or Natural Language Processing.

9. Working of the Chatbot

The chatbot begins by displaying a welcome message and instructions for the user. It then waits for user input in a continuous loop. Before processing the input, it converts all characters to lowercase and removes unnecessary spaces to ensure consistent matching.

The chatbot compares the user's input against predefined conditions using if-elif-else statements. If a matching condition is found, it displays the corresponding response. If the input is not recognized, it informs the user and suggests using the help command.

The conversation continues until the user types "bye", after which the chatbot displays a farewell message and terminates.

11. Challenges Faced

During the development of this project, one of the main challenges was handling different variations of the same question. This was solved by grouping multiple related user inputs into a single if-elif condition using the in operator. Another challenge was ensuring that user inputs were not affected by uppercase letters or extra spaces, which was addressed using the lower() and strip() string methods.
